##### 나의 풀이

```py
def solution(nums):
    answer = 0
    # set(nums) 활용
    s = set(nums)

    # len(nums)/2 활용
    if len(s) <= len(nums)/2:
        answer += len(s)
    else:
        answer += len(nums)/2
    
    return answer
```



##### 다른 풀이 1

```py
def solution(nums):
    # min 함수 활용
    return min(len(nums)/2, len(set(nums)))
```
