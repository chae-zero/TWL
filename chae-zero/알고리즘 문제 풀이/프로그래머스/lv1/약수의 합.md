### 약수의 합


##### 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/12928



##### 나의 풀이

```py
def solution(n):
    answer = 0

    for i in range(1,n+1):
        if n == 0:
            pass
        elif n % i == 0:
            answer += i
    return answer
```
