### 두 정수 사이의 합



##### 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/12912


##### 나의 풀이

```py
def solution(a, b):
    answer = 0
    # a=b이면, 둘 중 아무거나 리턴
    if a == b:
        return a
    # a,b의 대소관계는 정해져있지 않음
    elif a < b:
        for i in range(a,b+1):
            answer += i   
    elif b < a:
        for i in range(b,a+1):
            answer += i
        
    
    # a,b 사이 모든 정수의 합 리턴
    return answer
```



##### 다른 풀이 1

```py
def adder(a, b):
    # min, max 함수 활용
    return sum(range(min(a,b),max(a,b)+1))
```



##### 다른 풀이 2

```py
def adder(a, b):
    # a > b일 경우 a, b ->  b, a 로 치환
    if a > b: a, b = b, a

    return sum(range(a,b+1))
```



##### 다른 풀이 3

```py
def solution(a, b):
    return sum(range(a,b+1) if a <= b else range(b,a+1))
```
