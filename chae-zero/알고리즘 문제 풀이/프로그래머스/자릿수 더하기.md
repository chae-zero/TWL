### 자릿수 더하기



##### 문제 링크

https://school.programmers.co.kr/learn/courses/30/lessons/12931

##### 나의 풀이

```py
def solution(n):
    str_n = 0
    for i in str(n):
        str_n += int(i)
        
    return str_n
```



##### 다른 풀이 1

```py
def sum_digit(number):
    return sum([int(i) for i in str(number)])
    # 정수 리스트에 sum을 하면 원소의 합을 반환해줌
```
